Pengantar Hosting

Pengantar hosting adalah layanan yang menyediakan space (tempat) dan resource (sumber daya) di server internet agar website atau aplikasi Anda bisa diakses online. Ini adalah fondasi dasarnya.

Fungsi Utama Pengantar Hosting:

  • Menyimpan File atau Menyimpan semua file website (HTML, CSS, gambar, video, dll).
  • Menjalankan Aplikasi atau Menyediakan lingkungan untuk menjalankan skrip PHP, database MySQL, dll.
  • Menghubungkan ke Internet atau Memastikan server selalu terhubung ke internet dengan kecepatan tinggi.
  • Penyedia Alamat Email atauMenyediakan layanan untuk membuat akun email dengan nama domain Anda (contoh: nama@perusahaananda.com).

Jenis Control Panel Hosting Populer:

  • Shared Hosting - Seperti kos-kosan. Satu server dipakai banyak website. Murah, cocok untuk pemula.
  • VPS(Virtual Private Server) - Seperti rumah dalam kompleks (townhouse). Anda punya bagian server yang dedicated, lebih bebas dan kuat.
  • Cloud Hosting - Seperti menyewa ruang di pusat perbelanjaan yang fleksibel. Sumber dayanya bisa ditambah atau dikurangi sesuai kebutuhan.
  • Dedicated Server - Seperti membeli satu rumah lengkap. Satu server fisik hanya untuk Anda sendiri. Sangat kuat dan mahal.

cPanel - Control Panel

cPanel adalah control panel hosting web yang paling banyak digunakan di dunia. Ini menyediakan antarmuka grafis dan alat otomatisasi yang dirancang untuk menyederhanakan proses hosting website.

Fitur Utama cPanel:

  • File Manager - Mengelola file website tanpa FTP
  • Email Accounts - Membuat dan mengelola akun email
  • MySQL Databases - Membuat dan mengelola database
  • Domain Management - Menambahkan domain, subdomain, dan parkir domain
  • Softaculous Apps Installer - Instalasi aplikasi web dengan satu klik
  • Security - Konfigurasi SSL, IP Blocker, dan Hotlink Protection
  • Backup - Membuat dan memulihkan backup

Cara Mengakses cPanel:

Biasanya diakses melalui: http://yourdomain.com/cpanel atau http://server-ip:2082
